# 食药监局 CFDA 项目后端

## 介绍

本项目是为 [CFDA 前端](https://git.paperplane.cc/chiskat/cfda-app) 提供 API 接口而准备

## 配置项

初次启动必须进行配置，请参考 `.env.example`

## 本地开发

安装依赖：

```bash
npm i
```

开发模式运行：

```bash
npm run dev
```

## 编译部署

编译构建源码（输出到 `build/` 目录）：

```bash
npm run build
```

生产环境运行：

```bash
npm run prod
```
